LED Display Creator – User Guide

This tool is designed for embedded programmers who work with LED displays. It enables you to create static images and text that can be used on LED matrices, and offers three different export formats for seamless integration with your embedded projects. Additionally, collaboration is made simple through easy merging and importing features.

Getting Started

  • Launch the Application: Run the program (e.g., using python main.py or your favorite IDE).
  • The main window opens with a grid representing your LED display matrix. This grid serves as your canvas for designing images and text.

Designing LED Content

  • Paint Your Grid: Click individual cells to toggle them on or off. You can set colors to simulate multi-color LED displays.
  • Text Overlays: Use the text overlay tool to add static text. Customize the font, size, and style to match your LED display requirements.
  • Select & Edit: Click on row or column labels to select entire lines. Hold the Shift key to select a range of rows or columns for batch editing.
  • Image Import: Drag and drop an image file (PNG, JPEG, etc.) onto the window to convert it into an LED display pattern.

Export Formats

The tool provides three export formats to suit various embedded system needs:

  • Plain Format: Exports a simple binary grid, where each cell is either on (1) or off (0).
  • Formatted Format: Exports the grid in a grouped binary format for easier readability and integration.
  • Colored Format: Exports color-coded data, mapping the colors to a predefined palette, perfect for multi-color LED displays.

Choose the export format that best fits your target embedded system.

Collaboration

  • Merging Designs: Easily merge imported grid states with your current design, making team collaboration and iterative design simpler.
  • Import/Export: Share design files with colleagues using the provided import and export features to work on projects collaboratively.

Keyboard Shortcuts

Speed up your workflow using these shortcuts:

Shortcut Action
Ctrl+S Export the grid state.
Ctrl+O Import a saved grid state.
Ctrl+M Merge an external grid state with your current design.
Ctrl+R Reset the grid to its default (all cells off).
Ctrl+U Load a PNG image into the grid.
Ctrl+I Open the Text Overlay dialog to add or edit text.
Ctrl+C Copy the grid state in a formatted style to the clipboard.
Ctrl+Z Undo the last change.
Ctrl+Y Redo the last undone change.
Ctrl+W Set a global paint color for drawing cells.
Ctrl+F Change the color of all active (lit) cells.
Ctrl+E Change the color of cells with the picked color.
Ctrl+X Apply a predefined color palette to the grid.
Ctrl + Arrow Keys Shift the entire grid in the chosen direction.
Arrow Keys (no modifiers)
  • Shift selected rows or columns.
  • If both rows and columns are selected, only the intersection is affected.
P Activate the eyedropper tool to pick a cell’s color.
Escape Exit any active mode (painting, eyedropper, etc.).

Tips & Tricks

  • Multi-Selection: Hold the Shift key while clicking row or column headers to select a range.
  • Experiment: Use the undo/redo features to try different designs without losing your progress.
  • Collaborate: Share your exported design files with colleagues and merge changes easily using the merge import feature.
  • Optimize for LED Displays: Use the text overlay tool and color export options to tailor your design to your specific LED display hardware requirements.
